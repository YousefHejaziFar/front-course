.فونت یک عنصر را مشخص میکند font-family ویژگی
می تواند چندین نام فونت را font-family ویژگی نگه دارد fallback"به عنوان یک سیستم
اگر مرورگر فونت اول را پشتیبانی نمی کند فونت
بعدی را امتحان میکند
نام یک خانواده فونت مانند Family-name و غیره "times"، "courier"، "arial"
نام یک خانواده عمومی، مانند generic-family - "serif"، "sans-serif"، "cursive"، "fantasy"، "monospace".
با فونتی که می خواهید شروع کنید و همیشه با یک خانواده عمومی خاتمه دهید تا اگر فونت دیگری در دسترس نباشد مرورگر فونت مشابهی را در خانواده عمومی انتخاب کند
هر مقدار را با کاما جدا کنید

سبک قلم را برای یک متن مشخصfont-styleویژگی
میکند
Property Values:
-normal: The browser displays a normal font style. This is default

- italic: The browser displays an italic font style
- oblique: The browser displays an oblique font style
- initial: Sets this property to its default value.
- inherit: Inherits this property from its parent element.

  اندازه یک فونت را تعیین می کندfont-sizeویژگی
  Property Values:
  -medium: Sets the font-size to a medium size. This is default
  -xx-small: Sets the font-size to an xx-small size
  -x-small: Sets the font-size to an extra small size
  -small: Sets the font-size to a small size
  -large: Sets the font-size to a large size
  -x-large: Sets the font-size to an extra large size
  -xx-large: Sets the font-size to an xx-large size
  -smaller: Sets the font-size to a smaller size than the parent element
  -larger: Sets the font-size to a larger size than the parent element
  -length: Sets the font-size to a fixed size in px, cm, etc.
  -%: Sets the font-size to a percent of the parent element's font size
  -initial: Sets this property to its default value.
  -inherit: Inherits this property from its parent element.
